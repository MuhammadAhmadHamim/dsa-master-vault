## в—Ҳ Techniques in the Toolkit
```cpp
// Every pattern here has been used in a real solved problem.

// 1. Two Pointer вҖ” O(n) traversal without extra space
int left = 0, right = n - 1;
while (left < right) { /* close in */ }

// 2. Fast / Slow Pointer вҖ” Floyd's cycle + midpoint detection
ListNode* slow = head;
ListNode* fast = head;
while (fast && fast->next) { 
    slow = slow->next; 
    fast = fast->next->next; 
}

// 3. Dummy Head вҖ” clean edge case handling in linked lists
ListNode dummy(0);
ListNode* curr = &dummy;

// 4. In-Place Modification вҖ” O(1) space array operations
// Shift, overwrite, no auxiliary array needed

// 5. Pointer Reversal вҖ” iterative list restructuring
ListNode* prev = nullptr;
while (curr) { 
    ListNode* next = curr->next; 
    curr->next = prev; 
    prev = curr; 
    curr = next; 
}

// 6. Sliding Window вҖ” maintain valid range with queue
queue window;
window.push(new_element);
while (!window.empty() && window.front() < threshold) {
    window.pop();  // remove expired elements
}

// 7. Circular Buffer вҖ” modulo arithmetic for wraparound
rear = (rear + 1) % capacity;
front = (front + 1) % capacity;
// Reuses array space efficiently without shifting

// 8. Two-Stack Queue вҖ” LIFO to FIFO conversion
stack s1, s2;  // input stack, output stack
// Transfer s1вҶ’s2 reverses order, achieving FIFO

// 9. Position Tracking вҖ” round-based simulation
queue positions;  // store indices, not just values
positions.push(current_position + n);  // next round position
// Enables ordered processing across multiple rounds
```
